# Portfolio Project Images

This folder contains project images for the portfolio.

## Current Images

Using placeholder images from Unsplash via CDN:
- E-Commerce: Analytics dashboard
- Mobile App: Mobile devices  
- Brand Identity: Branding materials
- Corporate: Office workspace
- Photography: Camera equipment
- Dashboard: Data visualization

## To Add Your Own Images

1. Add images to this folder (recommended size: 800x500px)
2. Update src in index.html to point to local images
3. Example: `<img src="images/project-1.jpg" alt="Project Name">`

## Recommended Format

- Format: JPG or WebP
- Size: 800x500px (16:10 ratio)
- Quality: 80-85%
- File size: < 200KB per image
